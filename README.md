# Aertrip Flight Assignment

Live Demo:
https://aertrip-flight-assignment-pi7y.vercel.app/

## Tech Stack

- React JS (Vite)
- JavaScript
- CSS

## Features

- Sticky Header
- Flight Result Cards
- Sorting (Price, Duration, Departure, Arrival)
- Price Range Slider
- Clear Filters
- Dynamic Flight Count
- Error Handling

---

## How to Run the Project Locally

1. Clone the repository:
   git clone https://github.com/ShivaniSalwe77/aertrip-flight-assignment.git

2. Navigate to the project folder:
   cd aertrip-flight-assignment

3. Install dependencies:
   npm install

4. Start the development server:
   npm run dev

5. Open in browser:
   http://localhost:5173
